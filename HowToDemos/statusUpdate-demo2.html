Using jQuery and jQuizMe

jQuizMe StatusUpdate Demo 2

Goal:

Redirect the page to google.com if the user has answered more than 1 question correct when the quiz has finished.

Example:

Code:

var options = {};
options.statusUpdate = function( quizInfo, $quiz ){
	// The page will be redirect to a google if there is more than one question right on quit.
	if( quizInfo.numOfRight > 1 && quizInfo.hasQuit ){
		document.location = "http://www.google.com?q=I'm too smart";
	}
};
